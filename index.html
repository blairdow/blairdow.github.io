ABOUT ME

HI there! Im Blair, a front-end engineer who has a deep love for CSS, and also knows enough back end PHP/Laravel to be only mildly dangerous to your codebase. As a former designer, I love working closely with them, and am able to speak their language and understand design files in a way that eludes many other computer nerds out there. want your designers to fall in love with your new hire? it me.

In 2016, I built a clone of the classic Nokia game, Snake, with an 8-bit spin. I thought it would be a fun challenge to refactor it, and also a good way to show what I've learned in the last 7 years. My original version was pretty good for someone who was semi-new to coding, but there were definitely improvements to be made.

The main things were:

  • rewriting the project in vanilla JS so I could get rid of jQuery
  • actually making it work on mobile (ironically my original version only worked on desktop due to it using the arrow keys)
  • adding Webpack (definitely overkill for this game, but good practice)
  • adding Tailwind.css (not everyone's favorite way to CSS, but for me the benefits outweigh the drawbacks)

WORK

RUBY ON RAILS • ERB

Music Alert

Music Alert is an app that allows a user to create a list of songs, artists, and albums, and then sends a daily email with a reminder to listen to a random selection from their list. Perfect for those who keep forgetting to check out that new artist, or listen to that new album on the way home from work!

Built with Ruby on Rails on a PostgreSQL database, and ERB/Bootstrap/CSS frontend. Ruby Action Mailer and Cron task were used to automate emails.

NODE.JS • SOCKET.IO • MATERIALIZE CSS

Jeopardy with Friends - Live!

A chat based Jeopardy game using socket.io. This was a group project, built in 1 week. I was responsible for managing version control on Git, as well as the design. I also built a large part of the game/chat logic.

Built with Node.js, HTML/CSS, jQuery, Socket.io, Materialize.css, and the jservice.io API, which provided the Jeopardy questions.

IONIC MOBILE • NODE.JS • ANGULAR

FluffMates

An iOS mobile app using the Petfinder API to allow you to search for pets to adopt in your zip code. The idea came from a friend of mine who wanted "Postmates, but for adopting pets." Users can log in and select their three favorite pets, then click a button to have them delivered! (Next step is to hire a fleet of delivery drivers.)

Built with the Ionic mobile framework, Angular, HTML/CSS, Node.js and MongoDB.
