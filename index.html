In 2016, when I was just a baby engineer, I built a clone of the classic Nokia game, Snake, with an 8-bit spin. I thought it would be a fun challenge to refactor it, and also a good way to show off how my skills have improved in the last 7 years. My original version was pretty good for someone who was semi-new to coding, but there were definitely improvements to be made. (When are there not?)

The main changes were:

  • Rewriting the project in Vanilla JS so I could remove jQuery
  • Actually making it work on mobile (ironically my original version only worked on desktop due to it using the arrow keys)
  • Adding Webpack (definitely overkill for this game, but good practice)
  • Adding Tailwind.css (not everyone's favorite way to CSS, but for me the benefits outweigh the drawbacks)
